Project details in Brief

    [cite_start]
  • The Problem: Standard recipe apps are inflexible[cite: 26]. [cite_start]They don't let users adjust recipes for the ingredients they have or for different serving sizes[cite: 28].
    • [cite_start]
  • The Solution: "MealScale" is a smart Android app that personalizes cooking[cite: 30]. Users enter their desired servings, and the app scales the *entire* recipe.
  • Smart Scaling: The app's core feature. [cite_start]It automatically recalculates all ingredient quantities and intelligently adjusts the cooking duration based on the new serving size[cite: 32].
    • [cite_start]
  • Guided Cooking: Provides a clear step-by-step guide for all recipes and an integrated `CountDownTimer` for a confusion-free experience[cite: 33, 40, 41].
    • [cite_start]
  • Offline First: Built with Kotlin and XML, using a local SQLite database to store all recipe data, allowing 100% offline functionality[cite: 35, 36, 42].
    • [cite_start]
  • Project Goal: A minor project for the BCA-V Sem, demonstrating mobile development, database management, and modern UI/UX design[cite: 4, 34].
